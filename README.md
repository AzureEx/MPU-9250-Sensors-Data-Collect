# MPU-9250 (MPU-6500 + AK8963) I2C Driver in Python

MPU-9250 is a multi-chip module (MCM) consisting of two dies integrated into a single QFN package. One die the **MPU-6050** houses the 3-Axis gyroscope, the 3-Axis accelerometer and  temperature sensor. The other die houses the **AK8963** 3-Axis magnetometer. Hence, the MPU-9250 is a 9-axis MotionTracking device that combines a 3-axis gyroscope, 3-axis accelerometer, 3-axis magnetometer and a Digital Motion Processorâ„¢ (DMP).
